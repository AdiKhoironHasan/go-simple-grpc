# GO Simple gRPC

## Installation
Clone repository

```shell
git clone https://github.com/achjailani/go-simple-grpc.git
```
Install

```shell
cd go-simple-grpc && go mod download 
```

## Running Application
Run gRPC server
```shell
go run main.go grpc:start
```
Run HTTP Server
```shell
go run main.go
```
